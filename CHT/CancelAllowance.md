# 折讓單作廢上傳格式

### 範例格式
```
{
    "SellerID": "12345678",
    "AllowanceNumber": "AA12345678A001",
    "CancelDate": "2017-06-13T05:01:30",
    "CancelReason": "折讓單誤開",
    "Remark": "說明備註"
}
```
### 欄位說明
* SellerID: 賣方統編\(必須為e首發票客戶\)
* AllowanceNumber: 作廢折讓單號碼
* CancelDate: 作廢折讓單時間
* CancelReason: 作廢折讓單原因
* Remark: 作廢折讓單備註
